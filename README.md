# openclaw-memory-system
OpenClaw 摘要记忆系统：一键保存/读取，WSL 干净，历史存 D 盘
Startup Process:
1. Read SOUL.md for identity
2. Read USER.md for user info
3. Read MEMORY.md for long-term memory

Memory:
- Long-term: MEMORY.md

## 摘要操作

摘要存在 Windows D 盘：D:\小翎专属\聊天记录\摘要\
WSL 路径是：/mnt/d/小翎专属/聊天记录/摘要/

当用户说“存”时：
- 生成当前会话摘要，300字以内
- 文件名格式：主题-年月日-时分.md
- 保存到 /mnt/d/小翎专属/聊天记录/摘要/
- 回复“已存”

当用户说“取”或“取 [关键词]”时：
- 如果用户只说了“取”，列出最近5个摘要文件
- 如果用户说了“取 猫”，搜索文件名包含“猫”的文件
- 按修改时间倒序列出最近3个
- 只列文件名，不读内容
- 回复格式：“找到：1. 文件名1  2. 文件名2  读哪个？”

当用户说“读1”或“读2”或“读3”时：
- 读取对应的文件
- 回复“已加载，继续”

当用户说“列表”时：
- 列出摘要目录下所有文件
- 按修改时间倒序，显示最近10个
- 只列文件名，不读内容

当用户说“清空上下文”时：
- 先询问：“需要先保存摘要吗？”
- 用户确认后清空当前会话上下文
